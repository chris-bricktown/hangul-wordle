# 블루베리팜 — 산지직송 블루베리 쇼핑몰

블루베리를 홍보·판매하는 **백엔드 없는 정적 쇼핑몰**입니다. 순수
HTML/CSS/JavaScript로만 동작하며, 별도 서버나 결제 시스템이 없습니다.

## 실행 방법

설치 과정이 없습니다. `index.html`을 더블클릭해서 열거나, 가벼운 로컬
서버로 열어보세요.

```bash
# 그냥 파일 열기
open blueberry-shop/index.html     # macOS
start blueberry-shop/index.html    # Windows

# 또는 로컬 서버로
cd blueberry-shop
python -m http.server 8000   # http://localhost:8000 접속
```

GitHub Pages로 호스팅하려면 저장소 설정에서 Pages 소스를 이 브랜치의
`/blueberry-shop` 폴더(또는 루트로 이동시킨 뒤 루트)로 지정하면 됩니다.

## 구성

- `index.html` — 페이지 구조 (히어로, 상품, 농장 이야기, 후기, 문의)
- `style.css` — 전체 스타일
- `app.js` — 상품 데이터, 장바구니(localStorage), 주문 요청 모달 로직

## 백엔드 없이 "판매"가 동작하는 방식

결제·주문 서버가 없으므로, 장바구니에 상품을 담고 **주문 요청하기**를
누르면 주문 내용(상품명·수량·합계)을 정리한 텍스트가 만들어집니다.
이 내용을 아래 방법 중 하나로 보낼 수 있습니다.

- 📋 클립보드에 복사해서 카카오톡 등으로 전달
- 💬 `sms:` 링크로 휴대폰 문자 앱에 자동 입력
- 📞 `tel:` 링크로 바로 전화 연결

실제 운영 시에는 이 흐름을 거쳐 받은 주문을 확인하고, 입금 계좌나
실제 결제 수단(예: 카카오페이 송금 링크, 스마트스토어 연동 등)을
안내하면 됩니다.
