# 한글 꼬들 (Hangul Wordle)

세 글자 한글 단어를 10번 안에 맞히는 한글판 Wordle 게임입니다. 별도 빌드나
의존성 없이 **단일 HTML 파일**(`index.html`)로 동작합니다.

## 플레이 방법

1. `index.html`을 웹 브라우저에서 엽니다.
2. 세 글자로 된 한글 단어를 입력하고 **입력**(또는 Enter)을 누릅니다.
3. 색으로 표시되는 힌트를 보고 10번 안에 정답 단어를 추리합니다.

## 힌트 규칙

판정은 **두 단계**로 이뤄집니다.

### 1) 음절(글자) 단위 — 칸 배경색

| 색 | 의미 |
|----|------|
| 🟩 초록 | 글자와 **자리**가 모두 정답 |
| 🟨 노랑 | 단어에 **있지만 자리가 틀림** |
| ⬜ 회색 | 단어에 **없는 글자** |

### 2) 자모 단위 — 칸 아래 작은 힌트 바

각 글자 칸 아래의 작은 바 3칸은 **초성·중성·종성**을 각각 따로 채점합니다.
음절 전체는 틀렸더라도 자음·모음 일부가 맞으면 어디까지 근접했는지 알 수 있어,
일반 Wordle보다 더 촘촘한 힌트를 줍니다.

## 실행

설치 과정이 없습니다. 파일을 더블클릭하거나, 가벼운 로컬 서버로 열어도 됩니다.

```bash
# 그냥 파일 열기
start index.html      # Windows
open index.html       # macOS

# 또는 로컬 서버로
python -m http.server 8000   # http://localhost:8000 접속
```

## 구현 메모

- 순수 HTML/CSS/JavaScript, 외부 라이브러리 없음
- 한글 음절을 초/중/종성으로 분해하는 로직(`decompose`)으로 자모 단위 힌트 구현
- 정답 단어 풀은 `index.html` 내 `WORDS` 배열에서 관리 (현재 세 글자 단어 507개)
